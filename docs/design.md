# 基本設計

このアプリケーションは GitHub または外部サーバーからビルド済みの実行バイナリをダウンロードするための CLI ツールです。

## 入力

このアプリケーションは以下の入力パラメータを受け取ります。

| パラメーター名 | パラメーター名（短縮形） | 型                                                                  | デフォルト値 | 値の例                                       | 説明 |
| :------------- | :----------------------- | :------------------------------------------------------------------ | :----------- | :------------------------------------------- | :--- |
| `repo`         | `R`                      | 文字列。フォーマットは `OWNER/REPO` または `HOST/OWNER/REPO` です。 |              | `github.com/shibataka000/gh-release-install` |
| `tag`          | N/A                      | 文字列                                                              | N/A          |                                              |
| `pattern`      | N/A                      | マップ                                                              |              |                                              |
| `dir`          | `D`                      | 文字列                                                              | `.`          |                                              |

### `repo`

#### 概要

- GitHub リポジトリ名を指定します。

#### 型、フォーマット

- 文字列型です。
- フォーマットは `OWNER/REPO` または `HOST/OWNER/REPO` です。

#### デフォルト値

## 処理

### リリースアセットが GitHub でホストされている場合

1. 指定された GitHub リポジトリおよびタグからリリースアセットの一覧を取得します。
   1. GitHub リポジトリが予め定義した外部リリースアセットテンプレートの対象に含まれている場合、テンプレートからリリースアセットの一覧を生成します。
   2. それ以外の場合、GitHub API を利用してリリースアセットの一覧を取得します。
